Image 1: "What's On" page image: Events

Link to image 1 before edit
Link to image 1 after edit

For this image, I chose an image from unsplash.com, using the search terms "performance art". The goal was to subtly retouch the image so it would fit as a square on the "What's On" page.
To achieve this, I took the following steps/decisions:

  • The colour balance was retouched by using an adjustment layer on Affinity Photo, which made the image warmer and add a bit more coolness to the background and make the performer's faces look less washed out or yellow toned. The colour balance was shifted slightly towards Cyan, Magenta and Blue to achieve this.
  • The image was cropped to a square, focusing on the performers in the middle. This was necessary was the page where the image sits contains rows of squared images.
  • I saved it as a JPEG, as the quality did not need to be extremely high as it sat quite small within my mock-up (as it would on the live website), and it allowed for a smaller file size, which would result in a faster loading time on a real website. On the website, I would inlcude it as a progressive JPEG for accessibility and because it is an icon on the webpage rather than a whole image to be seen on a full page.
  • Final Specifications: 328 KB, 600 x 600
